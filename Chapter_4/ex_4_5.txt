Exercise 4.5: Determine the result of the following expressions.
(a) -30 * 3 + 21 / 5
(b) -30 + 3 * 21 / 5
(c) 30 / 3 * 21 % 5
(d) -30 / 3 * 21 % 4

Ans:
#include <iostream>
#include <climits>

int main()
{
std::cout <<-30 * 3 + 21 / 5<<std::endl; // Ans: -86
std::cout <<-30 + 3 * 21 / 5<<std::endl; // Ans: -18
std::cout<< 30 / 3 * 21 % 5<<std::endl;  // Ans: 0
std::cout << -30 / 3 * 21 % 4<<std::endl; // Ans: -2
    return 0;
}
