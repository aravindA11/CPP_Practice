 Exercise 6.32: Indicate whether the following function is legal. If so,
 explain what it does; if not, correct any errors and then explain it.
int &get(int *arry, int index) { return arry[index];
}
int main() {
int ia[10];
for (int i = 0; i != 10; ++i)
    get(ia, i) = i;
}

Ans: This is a valid program. It returns a reference to array element at specific index. 
The returned reference is assigned with the value of index number.
